Requisitos del Sistema

Asegúrate de que tu sistema cumpla con los siguientes requisitos mínimos:

  1. Sistema Operativo: Windows 10 / macOS / Linux
  2. Procesador: Intel Core i5 o equivalente
  3. Memoria RAM: 8GB
  4. Almacenamiento: 1GB de espacio disponible en disco
  5. Conexión a Internet

Instrucciones para la Descarga e Instalación

Sigue estos pasos para descargar e instalar el juego:

  1. Dirígete al sitio web oficial de Galactix Rex.
  2. Busca la sección de descargas y selecciona la versión compatible con tu sistema operativo.
  3. Haz clic en el botón de descarga y espera a que se complete.
  4. Una vez descargado, ejecuta el archivo de instalación y sigue las instrucciones en pantalla.
  5. Una vez completada la instalación, podrás iniciar el juego desde el menú de inicio o el acceso directo en tu escritorio.

Cómo Jugar

Una vez que hayas instalado el juego, sigue estos pasos para comenzar a jugar:

  1. Ejecuta Galactix Rex desde el menú de inicio o el acceso directo en tu escritorio.
  2. En la pantalla de inicio, selecciona tu perfil o crea uno nuevo.
  3. Elige tu modo de juego preferido: Modo Historia, Multijugador, o Desafío.
  4. Sigue las instrucciones en pantalla para controlar tu nave y esquivar los cometas.
  5. ¡Diviértete explorando el espacio y sobreviviendo a los desafíos que se te presenten!
